# The Liberating River Content Catalog
The Liberating River Content Catalog

This repository contains the official content catalog for The Liberating River™ Podcast and related media published under LiberatingLiving, LLC.

The catalog serves as the structured dataset that powers the Content Catalog page on the website and provides a centralized reference for all published media.

Purpose

The catalog tracks all publicly released content including:
	•	Podcast video episodes
	•	Podcast audio episodes
	•	Podcast clips
	•	YouTube Shorts
	•	Special standalone videos

It allows content to be indexed, filtered, and displayed on the website so listeners can easily explore the library of teachings and media.

⸻

Catalog Structure

The catalog is maintained as a structured CSV dataset.

Each row represents a single piece of published media.

Example structure:

Title            Media Type      Format            Series                           Pillar                  Month              Publish Date       Primary Link
Episode Title    Podcast         Podcast Video     The Liberating River Podcast     Spiritual Alignment     February 2026      2026-02-12         YouTube URL

⸻

Media Types

The catalog organizes media into the following categories:
	•	Podcast Audio (Podbean)
	•	Podcast Video (YouTube)
	•	Podcast Clip
	•	YouTube Short
	•	Special Video

These classifications allow the catalog to be filtered and navigated on the website.

⸻

Content Pillars

Content is organized according to the Liberating River teaching framework.

Typical pillars include:
	•	Spiritual Alignment
	•	Identity & Self-Worth
	•	Wealth Consciousness
	•	Perspective Shifts
	•	Divine Guidance

These pillars reflect the broader philosophy of Liberated Living™.
